# 📇 Contact Manager App

A simple contact management application built using Next.js.

---

## 🚀 Features (So far)

* Basic project setup using Next.js
* Global layout with Navbar
* Navigation bar with conditional rendering (login/logout)
* Mock backend using JSON Server

---

## 🛠️ Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* JSON Server

---

## ⚙️ Setup Instructions

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```

Run backend (JSON Server):

```bash
npm run server
```
